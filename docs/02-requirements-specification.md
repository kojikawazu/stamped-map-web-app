## 受け入れ基準

### F-001: スポット登録

- 地図上をクリックすると座標が取得され、登録モーダルが表示される
- 名前（必須）・カテゴリ（必須）・訪問日（必須）・メモ（任意）を入力して保存できる
- バリデーションエラー時はフィールド単位でエラーメッセージを表示する
- 保存後、地図上にマーカーが即座に表示される

### F-002: スポット一覧表示

- 登録済みスポットが左パネルにリスト形式で表示される
- 同時に右側の地図上にすべてのマーカーが表示される
- リスト項目クリックで地図がそのスポットにフォーカスする

### F-005: スポット削除

- 詳細表示から削除ボタンをクリック
- 確認ダイアログ表示（「本当に削除しますか？」）
- 確認後に削除実行、トースト通知で完了を表示

### F-006: 地図マーカー表示

- API Routes 経由で取得したスポットデータが地図上にマーカーとして描画される
- マーカーはカテゴリの色で表示される
- マーカークリックでスポット情報のポップアップが表示される

### F-008: ログイン

- ログイン画面（`/login`）でメールアドレスとパスワードでログインできる
- サインアップ機能は提供しない（アカウントは Supabase ダッシュボードで事前作成）
- Supabase 側で公開サインアップ（Enable sign-ups）を無効化する
- ログイン後、JWT トークンが発行される
- 認証エラー時はエラーメッセージを表示

### F-009: 認証ガード

- メイン画面ロード時にローディングシェルを表示
- クライアント側で getSession() により認証状態を判定
- 認証済み → メインコンテンツを表示
- 未認証 → /login にクライアント側リダイレクト
- 未認証のAPIリクエストには 401 が返る
- API が 401 を返した場合、セッションリフレッシュを試行し、失敗ならログイン画面へ

### F-010: カテゴリ管理

- デフォルトカテゴリ（食事・自然・観光・ショッピング・その他）が初期表示される
- ユーザーが新しいカテゴリを追加できる
- カスタムカテゴリに名前・色を設定できる
- カスタムカテゴリの編集・削除が可能（使用中スポットがある場合は警告）

### F-011: ソート機能

- 一覧の並び順を「訪問日順」「登録日順」から選択できる
- 昇順・降順の切り替えが可能
- デフォルト：訪問日の新しい順

### F-012: カテゴリフィルター

- カテゴリを選択して一覧・地図のスポットを絞り込める
- 複数カテゴリの同時選択が可能
- フィルター解除で全件表示に戻る

### F-013: ページネーション

- 一覧表示が一定件数（20件）ごとにページ分割される
- ページ切り替えUI（前へ / 次へ / ページ番号）が表示される
- ページネーションはリストのみ適用（地図は全マーカー表示）

### F-014: 名前検索

- 検索テキストボックスにスポット名を入力して検索できる
- 部分一致で検索される
- 検索結果は一覧・地図の両方に反映される
- 検索クリアで全件表示に戻る

### F-015: 地図位置記憶

- 地図の中心座標とズームレベルを localStorage に保存
- 次回アクセス時に前回の位置・ズームで表示
- 初回アクセス時は日本全体をデフォルト表示

### F-016: 空状態ガイド

- スポットが1件も登録されていないとき、ガイドメッセージを表示
- 「地図をクリックして最初のスポットを登録しましょう」等の案内

### F-017: エラー通知

- API失敗時にトースト通知でエラーメッセージを表示
- 通知は数秒後に自動で消える
- 登録・更新・削除の成功時にも成功トースト通知を表示

### F-018: ローディング表示

- データ取得中にローディングインジケータを表示
- 一覧：スケルトンUI or スピナー
- 地図：ローディングオーバーレイ

### F-019: ログアウト

- ヘッダー右上にログアウトボタンを配置
- クリックでセッション破棄 → ログイン画面にリダイレクト

## バリデーションルール

| フィールド | ルール |
|-----------|--------|
| スポット名 | 必須、1〜100文字 |
| カテゴリ | 必須、有効なcategory_id |
| 訪問日 | 必須、未来日は不可 |
| メモ | 任意、最大1000文字 |
| カテゴリ名（新規追加時） | 必須、1〜50文字、重複不可 |
| カテゴリ色（新規追加時） | 必須、有効なHEXカラーコード |
